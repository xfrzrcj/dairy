# **Beego --- golang web 框架学习**
## 介绍

本人主语言java。在接触golang后感觉效率挺高，相比java，编写多线程程序尤其方便。beego是个人在寻找Tomcat替代品时发现的一款不错的golang的web框架。框架上手简单，之前做的几个简单的api服务完全没问题,消耗资源相对tomcat来说少很多。目前在学习中，不定时更新学习进度。
